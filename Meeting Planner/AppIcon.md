# Meeting Planner App Icon Design

## Icon Concept
A modern, clean icon that combines the key elements of meeting planning and flight optimization.

## Primary Design Elements

### Main Symbol: Airplane + Pin Fusion
- **Central Element**: A stylized airplane silhouette merged with a location pin
- **Symbolism**: Represents travel planning and destination selection
- **Style**: Clean, geometric, minimal

### Supporting Elements
- **Background**: Gradient circle suggesting global connectivity
- **Accent**: Subtle grid pattern suggesting planning and organization
- **Color Scheme**: Blue (#007AFF) to Green (#34C759) gradient representing travel and success

## Technical Specifications

### Size Requirements (iOS)
- **1024x1024px**: App Store (required)
- **180x180px**: iPhone App (3x)
- **120x120px**: iPhone App (2x)
- **167x167px**: iPad Pro
- **152x152px**: iPad
- **76x76px**: iPad (1x)

### Design Guidelines
- **Corner Radius**: Automatic (iOS handles rounded corners)
- **Safe Area**: Keep important elements within 80% of icon bounds
- **Shadow/Effects**: None (iOS adds automatically)
- **Text**: No text in icon
- **Photorealism**: Avoid - use clean, symbolic design

## Color Palette

### Primary Colors
- **Blue**: #007AFF (iOS System Blue)
- **Green**: #34C759 (Success/Money Green)
- **White**: #FFFFFF (Highlights)
- **Gray**: #8E8E93 (Subtle details)

### Gradient Specifications
- **Background**: Radial gradient from #007AFF (center) to #0056CC (edge)
- **Airplane**: Solid white (#FFFFFF) with subtle shadow
- **Pin**: Gradient from #007AFF to #34C759

## Icon Variations

### Light Mode (Default)
- Bright blue-to-green gradient background
- White airplane with subtle gray shadow
- Clear, high contrast

### Dark Mode Compatible
- Slightly darker gradient background
- Maintain white airplane for visibility
- Ensure WCAG contrast compliance

## Symbolic Meaning
1. **Airplane**: Flight search and travel optimization
2. **Location Pin**: Meeting destination selection
3. **Gradient**: Journey from planning (blue) to success (green)
4. **Circular Background**: Global reach and connectivity
5. **Clean Lines**: Professional, efficient planning

## Implementation Notes

### Vector Format
Create in vector format (SVG/PDF) for scalability across all sizes.

### Testing Checklist
- [ ] Visible at 16x16px (Spotlight search)
- [ ] Clear at all required sizes
- [ ] Works on light and dark backgrounds
- [ ] Distinct from similar apps in App Store
- [ ] Represents app functionality clearly

### App Store Optimization
- Icon should be memorable and unique

## Final Recommendations
1. **Use Primary Concept**: Airplane + Pin fusion is most representative
2. **Maintain Simplicity**: Icon must read clearly at small sizes
3. **Follow iOS Guidelines**: Adhere to Apple's Human Interface Guidelines
4. **Test Extensively**: Verify visibility across all contexts and sizes
5. **Consider Seasonal Updates**: Subtle variations for major iOS updates

This icon concept effectively communicates the app's purpose while maintaining the clean, professional aesthetic expected in business productivity apps.
